# Installation (manual) [ENG]

1) Download the `BepInExPack` folder.
2) Move the contents of `BepInExPack` folder except for the file `README.md` into the game folder (where the game executable is located).
4) Run the game. If everything runs correctly, you will see BepInEx console pop up on your desktop.

Link to the mod: https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/

# Установка (инструкция) [RUS]

1) Скачайте папку `BepInExPack`.
2) Перенесите все файлы кроме `README.md` в корневую папку с игрой.
3) Запустите игру. Если вы все сделали правильно, то вы увидите консоль BepInEx на вашем рабочем столе.

Ссылка на мод: https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/
