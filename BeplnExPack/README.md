# Installation (manual)

1) Install `BepInExPack` folder into your downloads folder.
2) Move the contents of `BepInExPack` folder into the game folder (where the game executable is located)
   (It is not necessary to transfer the `README.md` to the game folder.).
4) Run the game. If everything runs correctly, you will see BepInEx console pop up on your desktop.

Link to the mod: https://thunderstore.io/c/repo/p/BepInEx/BepInExPack/
