# The repository of mods for R.E.P.O. (ENG)
Hi, friend! This repo contains all the mods I use to play the famous game R.E.P.O. As soon as a new version of the mods comes out, I update it right away. Only the freshest files are kept here. Also, the entire repository is written in two languages: ENG and RUS.

---

## Repository structure

The repository has two main areas:

- `BepInExPack/` - the folder with the main mod `BepInEx` for the correct operation of most mods and mods-lib
- `plugins/` - a folder with all the mods that I use

In more detail:

- `BepInExPack/`
  - `README.md`
  - `.doorstop_version`
  - `doorstop_config.ini`
  - `winhttp.dll` - the main BepInEx mod file
  - `BepInEx/` - the main BepInEx mod folder, which contains the config/, plugins/ folders, etc.

- `plugins/`
  - `README.md`
  - `mod1/`, `mod2/`, `mod3/` ... - separate folders for mods, each containing lots of files for proper mod functionality
  - one main file with the same name, for example:
    - `plugins/MoreHead/MoreHead.dll`
    - `plugins/WesleysEnemies/WesleysEnemies.dll`
    - and so on.
  - `IncompatibleMods/` - a folder where mods from the `plugins/` folder that are incompatible with the latest R.E.P.O. version will be placed.

---

## Issues, questions, etc.

If you notice a mod isn't working or there's a new version of a mod but the repo info hasn't been updated yet, don't get upset—I'm probably already fixing that issue or looking for a suitable mod to replace the one incompatible with the latest R.E.P.O. version. Also, the `plugins/README.md` file will have all the mod info in two languages: ENG and RUS.
Check the repo every 2 weeks if possible).

P.s.: All mods are taken from this website: https://thunderstore.io/c/repo/

# Репозиторий модов для R.E.P.O. (RUS)

Привет, друг! Здесь собраны все моды, с которыми я играю в знаменитую игру R.E.P.O. Как только выходит новая версия модов, я сразу же её обновляю. В этом репозитории лежат только самые свежие файлы. Также, весь репозиторий написан на двух языках: ENG и RUS.

---

## Структура репозитория

В репозитории предусмотрены две основные области:

- `BepInExPack/` - папка с основным модом `BepInEx` для корректной работы большинства модов и мод-либов.
- `plugins/` - папка со всеми модами, которые я использую.

Подробнее:

- `BepInExPack/`
  - `README.md`
  - `.doorstop_version`
  - `doorstop_config.ini`
  - `winhttp.dll` - основной файл мода BepInEx.
  - `BepInEx/` - основная папка мода BepInEx, в которой будет лежать папка config/, plugins/ и т.д.

- `plugins/`
  - `README.md`
  - `mod1/`, `mod2/`, `mod3/` ... - несколько отдельных папок для модов, которые содержат много файлов для корректной работы мода.
  - один основной файл с таким же именем, например:
    - `plugins/MoreHead/MoreHead.dll`
    - `plugins/WesleysEnemies/WesleysEnemies.dll`
    - и т.д.
  - `IncompatibleMods/` - папка, в которой будут лежать моды из папки plugins/ несовместимые с последней версией R.E.P.O.

---

Также в файле `plugins/README.md` будет представлена вся информация по модам на двух языках: ENG и RUS.
